What is it?

This small but might really useful jQuery Plugin makes possible to push elements into specific containers depended on certain media queries.

Possible use cases:

  • Bring spread elements together to display them inside offcanvas
  • Rearrange elements for better usability and reading on smaller devices
  • Trick the natural behavior of order and wrap to avoid from doing thaumaturgic CSS feats
  • And many others...

Installation

Just include the pushme.js and initialize it for your elements. Note: Also make sure that jQuery is defined.

<script type="text/javascript" src="path/to/your/pushme.js"></script>

$(window).on('resize', function() {
    $('.foo').pushme({
        element: '.bar',
        pushAction: 'prependTo',
        mq: 'screen and (max-width:767px)'
    });
});
